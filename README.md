## 🚀 Featured Projects

### 🧠 EEG-Based Brain-Computer Interface — Motor Imagery Classification
> *Enabling communication and control for patients with ALS, stroke, and severe motor disabilities*

Acquired multi-session EEG signals using OpenBCI hardware from subjects performing left/right hand motor imagery tasks. Applied a complete signal processing pipeline — bandpass filtering (8–30 Hz), ICA artifact removal, Common Spatial Pattern (CSP) feature extraction — and trained an SVM classifier achieving ~78% accuracy. Includes a real-time Unity-based neurofeedback game used for subject engagement and validation.

**Stack:** Python · MNE · scikit-learn · OpenBCI SDK · Unity  
**Impact:** Prototype assistive control system for paralysed patients  

[![View Project](https://img.shields.io/badge/View%20Project-GitHub-black?style=flat&logo=github)](https://github.com/purohith08/BCI-IV-YEAR-project)

### 🩺 SmartPatch — Wearable Continuous Health Monitor
> *Post-discharge cardiac monitoring for patients who can't afford hospital stays*

Designed a low-power wearable patch integrating AD8232 ECG front-end and MAX30102 pulse oximeter (SpO2 + HR) sensors, with BLE wireless transmission to a companion mobile application. Targets continuous cardiac monitoring for post-hospitalization and elderly patients, reducing re-admission rates through early anomaly detection.

**Stack:** ESP32 · AD8232 · MAX30102 · BLE · Flutter  
**Impact:** Addresses the gap in affordable remote patient monitoring  

[![View Project](https://img.shields.io/badge/View%20Project-GitHub-black?style=flat&logo=github)](https://github.com/purohith08/SmartPatch)

---

### 🚁 Medical Supply UAV — Last-Mile Healthcare Delivery
> *Autonomous drone for delivering critical medicines and blood units to remote regions*

Engineered an autonomous UAV with HMC5883L magnetometer-based navigation, real-time WiFi telemetry, and custom flight stabilization firmware on Arduino. Conceptualized for delivering time-critical medical supplies — blood, vaccines, emergency drugs — to areas inaccessible by road. Project developed in collaboration with ISRO technology guidelines.

**Stack:** Arduino · C++ · HMC5883L · ESP8266 WiFi · IMU  
**Impact:** Estimated 40–60% reduction in critical supply delivery time to remote areas  

[![View Project](https://img.shields.io/badge/View%20Project-GitHub-black?style=flat&logo=github)](https://github.com/purohith08/DRONE-TECHNOLOGY)
